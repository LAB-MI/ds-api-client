# NodeJs

## Version 
| | version |
|--|--|
| Nodejs | v16.14.2 |
| npm | 8.5.0 |


## Objectif

Export sous le format csv  les instructeeurs et les dossiers d'une démarche

## usage

```
npm install
mkdir out

EXPORT TOKEN_DEMARCHE=<token>
node server.js "numero de demarche"
```
Les fichiers csv seront généré dans le répertoire out
